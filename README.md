# IHL Python Class - Homework Repository

This is the repository I'll be using to share homework assignments and class materials throughout the course.

The `main` branch is where all assignments will be posted. Each of you has your own branch — that's where you'll do your work and submit it. Do not push anything to `main`.

## Getting started

Clone the repo and switch to your branch:

```bash
git clone git@github.com:<repo-url>.git
cd <repo-name>
git checkout <your-branch>
```

## For every homework

Pull from main to get the latest assignment, do your work, then push:

```bash
git pull origin main
git add .
git commit -m "Homework #X"
git push origin <your-branch>
```

That's it. If something is not working, reach out before the due date.
